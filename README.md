# FL-Blockchain Reward Distribution

Đề tài: **Cải tiến cơ chế phân phối phần thưởng dựa trên đóng góp đa chiều trong hệ thống Học máy Liên kết kết hợp Blockchain**

Sinh viên: Phan Hồng Đạt — 23520266 | GVHD: ThS. Nguyễn Khánh Thuật

## Cài đặt nhanh
```bash
python3.11 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
npm install && npm run compile
npx hardhat node &   # Terminal riêng
npm run deploy && npm run fund
pytest tests/ -v
python experiments/run_experiment.py --dataset mnist --scenario K1 --config C --alpha 0.5
```
